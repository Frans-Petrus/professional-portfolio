# Professional Portfolio (Virtual CV)

## Overview

This project is a professional portfolio website (Virtual CV) developed as part of a university web development project.

The website presents my education, work experience, technical skills, projects, achievements, volunteer experience, and professional contact information in a modern, responsive, and user-friendly format.

Unlike a traditional CV, this portfolio provides additional detail while remaining easy to navigate and visually appealing.
